# 🌸 Portfólio Pessoal — Vitória

Portfólio pessoal desenvolvido como projeto acadêmico no curso de Bacharelado em Engenharia de Software — Uninter (2026).

## 🔗 Acesso

[Ver projeto online]((https://vitoriaalves00.github.io/portfolio-pessoal/index.html))

## 💻 Sobre o projeto

Site multipágina com identidade visual própria: tema dark, acentos em rosa e efeito de brilho nos cards. Desenvolvido sem frameworks ou bibliotecas de UI — apenas HTML, CSS e JavaScript puro.

**Páginas:**
- **Home** — apresentação pessoal com foto de perfil
- **Sobre mim** — quem sou, habilidades e motivações
- **Formação** — trajetória acadêmica e idiomas
- **Portfólio** — projetos desenvolvidos
- **Contato** — formulário funcional com validação

## 🛠️ Tecnologias

- **HTML5** — estrutura das páginas
- **CSS3** — estilização, responsividade e arquitetura multi-arquivo
- **JavaScript** — validação de formulário e integração com EmailJS
- **EmailJS** — envio de e-mails sem backend
- **Canva** — criação do background personalizado
- **Google Fonts** — fontes Bitcount Prop Double, Dancing Script e Roboto Mono

## ✨ Funcionalidades

- Layout responsivo para mobile e desktop
- Formulário de contato com validação em tempo real e feedback via toast
- Identidade visual consistente em todas as páginas
- Navegação entre páginas sem frameworks


## 📚 Contexto acadêmico

Projeto avaliativo da disciplina de Aplicação Web e Móveis — Engenharia de Software, Uninter, 2026.
